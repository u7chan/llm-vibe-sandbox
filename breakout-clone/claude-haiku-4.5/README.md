# Breakout Clone

A classic Breakout arcade game implementation in HTML5 Canvas with JavaScript.

## Features

- **Paddle Control**: Use left/right arrow keys to move the paddle
- **Ball Physics**: Realistic ball collision with walls, paddle, and bricks
- **Brick Layout**: 45 colorful bricks arranged in 5 rows
- **Score Tracking**: Gain 10 points for each brick destroyed
- **Lives System**: Start with 3 lives; lose one when the ball falls below the paddle
- **Win/Lose Conditions**: Break all bricks to win, or run out of lives to lose

## How to Run

1. Open `index.html` directly in any modern web browser
   - Simply double-click the file, or
   - Right-click and select "Open with" → your browser

## Controls

| Key | Action |
|-----|--------|
| **←** / **→** | Move paddle left/right |
| **Space** | Launch ball/start game |
| **New Game** button | Reset and start over |

## Gameplay

1. Click "New Game" or wait for the game to start
2. Press **Space** to launch the ball
3. Use **arrow keys** to move the paddle and keep the ball in play
4. Break all bricks to win
5. If the ball falls below the paddle, you lose a life
6. Lose all 3 lives and it's Game Over

## Game Stats

- **Score**: Points earned from breaking bricks (10 per brick)
- **Lives**: Remaining attempts
- **Bricks Left**: Number of active bricks remaining

## Brick Colors

- **Red**: Top row (worth 10 points)
- **Yellow**: Second row (worth 10 points)
- **Green**: Third row (worth 10 points)
- **Cyan**: Bottom rows (worth 10 points)

Enjoy!
